# setup
My DEV Setup

### How run git_setup
Execute this follow steps
```sh
$ chmod +x git_setup.sh
$ ./git_setup.sh
```
after run this close the terminal and open a new one.

#### Git alias Commands:
- git sw = git switch
- git l1 = git log --oneline

#### Bash alias for Git:
- gitfep = git fetch and git pull
